What is JPortAudio?

JPortAudio is a Java wrapper for PortAudio. It provides audio I/O for Java applications. It can be used as an alternative to JavaSound. Note that the JPortAudio API is not JavaSound compatible. The JPortAudio API is very close to the PortAudio API.

Installing on Mac

To use JPortAudio with a single application:

  1. Add jportaudio.jar to your CLASSPATH,
  2. Place libjportaudio.jnilib in your library path.

To use JPortAudio with any Java application:

  1. Place the jportaudio.jar and libjportaudio.jnilib files in your "~/Library/Java/Extensions" folder.
